# natural-deduction-prover

A web-based natural deduction proof generator for propositional and predicate logic.

This prover currently does not support identity (`=`), function symbols, or arithmetic operators.

This project was developed with reference to Sieg & Byrnes, *Normal Natural Deduction Proofs*.

## Features

- Propositional logic proof search
- Predicate logic proof search
- Fitch-style proof output
- In-browser execution

## Usage

Enter premises and a conclusion, then click **Proof**.
